# Lena Parks - Parktrainer

Eine interaktive Web-Anwendung zum Üben des Parkens mit realistischer Physik und Tutorials zu deutschen Verkehrsregeln.

## Features

- 2D-Parksimulator mit Matter.js Physik-Engine
- Geführte Tutorials für verschiedene Parksituationen
- Deutsche Verkehrsregeln integriert
- Responsive Design

## Installation

1. Abhängigkeiten installieren:
   ```bash
   npm install
   ```

2. Entwicklungsserver starten:
   ```bash
   npm run dev
   ```

3. Öffnen Sie Ihren Browser und gehen Sie zu `http://localhost:5173`

## Verwendung

- **Startseite**: Übersicht über die Anwendung
- **Simulator**: Freies Üben des Parkens
- **Tutorials**: Geführte Lektionen mit Erklärungen

## Technologien

- React
- TypeScript
- Vite
- Matter.js (Physik-Engine)
